BUILD A COMPLETE TELEGRAM ORGANIC MARKETING & LEAD GENERATION AUTOMATION SYSTEM. This is a dual-purpose system: (1) used internally for WeMagnifAI to get clients, (2) sold as a SaaS product. Build everything described below WITHOUT SKIPPING anything:

--- PART 1: TELEGRAM BOT ENGINE (Python/Node.js in /telegram-engine/ folder) ---

Create /telegram-engine/ with these files:

1. /telegram-engine/README.md - Full documentation
2. /telegram-engine/package.json (or requirements.txt) - All dependencies
3. /telegram-engine/config.example.json - Config template:
   - telegram_api_id, api_hash, phone, bot_token
   - target_keywords: ["AI agency", "digital marketing", "automation"]
   - niche: "AI Growth Agency"
   - website_url, cta_message
   - daily_message_limit: 50
   - daily_group_join_limit: 10
   - human_delay_min_seconds: 45
   - human_delay_max_seconds: 180
   - openai_api_key

4. /telegram-engine/core/bot_engine.js (or .py) - Main orchestrator that runs all modules on schedule

5. /telegram-engine/modules/group_discovery.js - 
   - Uses Telegram API to search groups by keyword
   - Scores groups by member count, activity level, relevance
   - Filters out spam groups
   - Returns ranked list of target groups
   - Stores discovered groups in groups_db.json

6. /telegram-engine/modules/group_joiner.js -
   - Joins top-scored groups respecting daily limits
   - Adds random delays (human-like)
   - Tracks join history to avoid re-joining
   - Implements cooldown periods

7. /telegram-engine/modules/content_generator.js -
   - Uses OpenAI GPT-4o to generate platform-native Telegram content
   - Content types: value posts, tips, mini case studies, polls, questions, resource shares
   - Generates content tailored to group's niche/topic
   - Adds natural CTA that mentions the website/service
   - Content is varied to avoid detection
   - Templates: 5 different formats that rotate

8. /telegram-engine/modules/group_poster.js -
   - Posts AI-generated content to joined groups
   - Schedules posts at optimal times
   - Tracks which groups were posted to and when
   - Respects group posting rules (reads pinned messages)
   - Implements per-group cooldown (don't spam same group)

9. /telegram-engine/modules/lead_scraper.js -
   - Scrapes active members from target groups
   - Qualifies leads by: bio keywords, activity recency, group memberships
   - Scores each lead 1-10 on fit for the niche
   - Stores leads in leads_db.json with full profile data
   - Filters out bots, inactive accounts, already-contacted users

10. /telegram-engine/modules/dm_outreach.js -
    - AI-powered personalized DM generation using lead's profile data
    - 3-message sequence: intro -> value -> CTA
    - Conversation AI that responds to replies intelligently
    - Tracks conversation state per lead
    - Respects daily DM limits to avoid spam flags
    - Has fallback responses for common objections

11. /telegram-engine/modules/comment_engager.js -
    - Monitors joined groups for relevant discussions
    - AI generates helpful, value-adding replies to posts
    - Positions the account as an expert/authority
    - Subtly mentions website/service when contextually appropriate
    - Tracks engagement history

12. /telegram-engine/modules/analytics_tracker.js -
    - Tracks all actions: groups joined, posts sent, DMs sent, replies received
    - Calculates: groups_discovered, groups_joined, posts_published, leads_scraped, dms_sent, replies_received, conversion_rate
    - Saves daily reports to /data/reports/
    - Outputs JSON for dashboard consumption

13. /telegram-engine/data/ - Folder with:
    - groups_db.json (empty template)
    - leads_db.json (empty template)
    - contacted_db.json (empty template)
    - analytics.json (empty template)

14. /telegram-engine/safety/rate_limiter.js -
    - Enforces all daily limits
    - Implements exponential backoff on errors
    - Detects and handles flood waits from Telegram
    - Rotates sessions if multiple accounts configured
    - Anti-pattern detection (varies message lengths, timing)

15. /telegram-engine/scheduler.js -
    - Cron-based scheduler for all tasks
    - Morning: group discovery + join new groups (8am)
    - Mid-morning: post content to groups (10am)
    - Afternoon: scrape leads + send DMs (2pm)
    - Evening: engage with comments (6pm)
    - Night: generate analytics report (10pm)
